## General Info

Service for sending emails.

## Technologies
- Java 17
- Spring Boot 3
- Spring Boot Data
- Elasticsearch
- RabbitMQ
- Kibana
- JUnit 5, Mockito, Testcontainers
- Lombok
- Maven
- Docker

## Set-up
1. Create network: ```docker network create email-service-network```
2. Run docker compose: ```docker-compose up -d```
